# Technical Drafting 📐

Storage for blueprints, CAD exports, and architectural diagrams.

- **Display Logic**: Optimized for the `draft-grid` using a **Wide (5:3)** object-fit within a 250px container.
- **Visual Style**: Uses a CSS filter of `grayscale and contrast` to create a professional "blueprint" aesthetic.
- **Hover State**: Filters are removed on hover to reveal the true colors of the drafting work.
