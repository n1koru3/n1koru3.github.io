# Traditional Art Gallery 🖼️

Contains high-fidelity images of physical paintings and sketches.

- **Display Logic**: Optimized for the `portrait-row` class using an **aspect-ratio: 4 / 5**.
- **Interactivity**: 
  - Supports the **Lightbox** script for full-screen viewing.
  - Hovering triggers a **Neon Purple (`--secondary`)** border glow and a translateY lift of -8px.
