# Other Creative Works 🌀

A collection of miscellaneous creative projects, experiments, and "Other Works" showcased in the ribbon carousel.

- **Display Logic**: Optimized for the `other-ribbon` carousel wrapper.
- **Aspect Ratio**: These cards are designed with a flexible height of **250px** (Desktop) and **200px** (Mobile) to allow for quick swiping/scrolling.
- **UI Element**: Includes a **Purple Themed Caption** overlay (`carousel-caption`) that highlights the project title in Mint Green.
- **Adaptive**: The layout automatically shifts from 5 items per row on Desktop to 1.2 items on Mobile to encourage swiping.
