# Software Systems 💻

Visuals for technical projects and software applications.

- **Display Logic**: These assets function as background layers for the `soft-card` elements.
- **UI Treatment**: Images are processed with `brightness and grayscale` to ensure high contrast for the text overlays and project tags.
- **Interactivity**: Hovering triggers a scale increase of `1.05` and an orange (`#FFA500`) glow.
