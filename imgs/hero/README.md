# Hero Assets 🌌

This folder contains the core visual identity for the landing page.

- **Purpose**: Provides the backdrop for the "Glitch" name effect and the interactive glow sphere.
- **Adaptive Strategy**: 
  - `bg.jpg`: Desktop background with a right-side focus to allow left-aligned text.
  - `bg-mobile.jpg`: Vertically oriented background to prevent stretching on mobile devices.
- **Visuals**: Works in tandem with the CSS `glow-sphere` radial gradient (`var(--primary)` to `var(--secondary)`).
