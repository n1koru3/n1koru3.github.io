# Digital Art & Illustrations 🎨

This folder contains your digital paintings, character designs, and vector illustrations.

- **Display Logic**: These assets use the `art-card` class and are often grouped in the `portrait-row` or `square-row` layouts.
- **Styling**: Features a **Neon Purple (`--secondary`)** border glow and a smooth lift effect on hover.
- **Interactive**: Fully compatible with the **Lightbox** zoom-in feature for detail viewing.
